# 🎮 Ultimate 2048 Game 🚀
The classic 2048 game made with HTML, CSS, and JavaScript. 😎
Challenge yourself to reach 2048 (or more)! 🎯

# 🚀 How to Play
Play the game in your browser (PC only):

Combine numbers to reach 2048. Good luck! 🍀
